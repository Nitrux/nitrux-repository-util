# Nitrux repository utility.

A tool to manage our Debian repositories with aptly.

usage :

```
  wget -qO - \
  	https://raw.githubusercontent.com/Nitrux/nitrux-repository-util/master/upload-pkgs \
  	| sh -s <files>

```
